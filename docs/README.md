# Catalance Documentation

Welcome to the Catalance project documentation. Catalance is an AI-powered managed freelance marketplace.

## System Documentation
This `docs` directory contains comprehensive breakdowns of the platform's core logic, generated from source code analysis.

### 1. [Business Model & Product Workflows](./BUSINESS_MODEL.md)
Contains the exact mechanism of the multi-sided marketplace:
- The AI Partner (Client Intake pipeline)
- Intelligent Freelancer Matching mechanics
- Revenue Model (Upfront payments, Escrow, and 30% Platform Fee)
- Project Manager operational guidelines

### 2. [Technical Architecture](./TECHNICAL_ARCHITECTURE.md)
Contains details strictly about the codebase structure and setup:
- Database Schema layout (Prisma)
- Backend Infrastructure details (Controllers, Services)
- Frontend Architecture (Dashboards, UI layer, Routing)

## Quick Start (Development)
The project is split into a frontend application and a Node.js backend. See the main `README.md` in the repository root for immediate terminal commands to start the Vite and Express servers.
